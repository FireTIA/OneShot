# Settings

Этот файл описывает настройки OneShotPin by FireSoft.

---

## `OS_Target`

**Возможные значения:**

- `NetHunter`
- `Kali`

**Описание:**

Определяет, куда сохранять итоговые файлы (логи и результаты проверки сетей):

- при `NetHunter` путь, например: `/sdcard/nh_files/OneShotPin_Log`
- при `Kali` путь, например: `/home/kali/OneShotPin_Log`
- при других значениях используется путь по умолчанию (например `/root/OneShotPin_Log`)

---

## `Hide_Password_AP`

**Возможные значения:**

- `True`
- `Half`
- `False`

**Описание:**

Управляет тем, как пароль точки доступа отображается в выводе скрипта:

- `True` — полностью скрывать пароль.
- `Half` — показывать только часть пароля (остальное маскируется).
- `False` — показывать пароль полностью.

## `Change_MAC_WlanX_Startup`

**Возможные значения:**

- `True` — псевдослучайный MAC, **с сохранением заводского OUI**  
  (используется `macchanger -a`).
- `"True-r"` — полностью случайный MAC, **меняется и OUI**  
  (используется `macchanger -r`).
- `False` — не менять MAC-адрес.
- `"<ваш MAC>"` — строка с кастомным MAC-адресом, например `"00:11:22:33:44:55"`  
  (используется `macchanger --mac=<ваш MAC>`).

**Описание:**

Определяет поведение модуля WlanX при старте скрипта:

- если значение отлично от `False`, перед запуском основной логики интерфейс `wlanX`
  временно выключается, на него применяется выбранная схема смены MAC, затем интерфейс снова включается.

## `Macchanger_Output_Mode`

**Возможные значения:**

- `"raw"` — Выводит вывод Macchanger 
- `"pretty"` — Выводит в красивом стиле с форматированием

**Описание:**

Тип вывода информации о работе Macchanger:

- Да

---

## `Hello_banner`

**Возможные значения:**

- `"default"` — полный приветственный баннер:
  название скрипта, автор перевода, GitHub, версия и т.д.
- `"classic"` — укороченный баннер с основной информацией о скрипте.
- `"compact"` — компактный баннер в одну строку: имя скрипта и короткая версия.

**Описание:**

Определяет стиль приветственного баннера, который выводится в начале работы скрипта.
